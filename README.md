# 🌐 My Portfolio

**My Portfolio** is a personal website built with **Next.js**, **TypeScript**, and **Tailwind CSS**, and deployed via **Vercel**. It serves as an interactive and responsive platform to showcase my professional journey, experience, and skills.

---

🛠️ **Tech Stack**

- ⚛️ **Next.js** – Framework for building fast, SEO-friendly React apps  
- 🔐 **TypeScript** – Type-safe codebase for improved reliability  
- 🎨 **Tailwind CSS** – Utility-first CSS for clean, responsive design  
- ☁️ **Vercel** – Deployment platform optimized for Next.js projects

---

📌 **Main Sections**

- 🏠 **Home** – Brief intro with a professional summary
- 👋 **About Me** – A short narrative of my background, interests, and goals  
- 💼 **Experience** – Timeline of roles and responsibilities  
- 🎓 **Education** – Academic background and achievements  
- 🧠 **Skills** – Technical skills and tools I use  
- 📬 **Contact** – Direct links or form to get in touch

---

✨ Built with a clean UI, smooth animations, and mobile-first responsiveness to create a strong online presence and highlight my work and background professionally.
